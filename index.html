Html Cheat Sheet

Name Syntax Description
Doctype <!DOCTYPE html> Defines the document to be HTML5
HTML <html> The root element of an HTML page
Head <head> Contains meta information about the document
Title <title> Specifies a title for the document
Body <body> Contains the visible page content
H1, H2, H3,..H6 >h1>, <h2>, <h3>,..<h6> Defines a large heading(h1 is the biggest heading and then it decreases from their on.)
Paragraph <p> Defines a paragraph
